
## Chrome extension for cleaner Facebook design (*Inspired by Twitter's feed)

- no sidebars
- flat desing (no shadows, no border-radiuses) 
- no ticker
- bigger and centered video player 
- only posts feed
- no list borders
- mobile screen size not supported yet, min width ~700px
